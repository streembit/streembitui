Open source


The system is free, fully open source and released under the Apache 2.0 license. The source code is available at https://github.com/Streembit. Users can download the source and build the application from directly the source code. We encourage peer review by security experts and software developers to verify there are no security backdoors exist in the system and the application complies with security and communication standards (FIPS, JWT, JWS, JWE, etc).
